# face-recognition-dd

### Installation:

#### 1) Go to directory

```bash
cd face-recognition-dd/
```

#### 2) Installing the virtual environment

```bash
python3 -m venv venv
```

#### 3) Activating the virtual environment

##### on Linux

```bash
source venv/Scripts/Activate
```

##### on Windows

```bash
venv/Scripts/Activate
```

#### 4) Installing dependencies:

```bash
pip3 install -r requirements.txt
```
